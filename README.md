# Sales & Service Analytics Dashboard

## Project Overview

This project demonstrates the creation of business intelligence dashboards for an automotive dealership network using Microsoft Excel and IBM Cognos Analytics.

The objective was to analyze dealership sales performance, profitability, customer sentiment, and vehicle recall information through interactive dashboards and visualizations.

## Business Problem

Regional managers require a consolidated view of sales and service performance across multiple dealerships. Manual analysis of large datasets is time-consuming and limits timely decision-making.

## Solution

Designed and developed Sales and Service dashboards to provide actionable insights into dealership operations and performance.

## Key Features

### Sales Dashboard

* Total Profit KPI
* Quantity Sold KPI
* Average Quantity Sold KPI
* Quantity Sold by Model
* Profit by Dealer ID

### Service Dashboard

* Vehicle Recall Analysis
* Customer Sentiment Analysis
* Sales vs Profit Trend Analysis
* Recall Heat Map
